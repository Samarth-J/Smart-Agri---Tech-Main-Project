# 🌱 Smart Agri-Tech Main Project

An **AI-powered Smart Agriculture Platform** that helps farmers make data-driven decisions for crop planning, yield prediction, fertiliser usage, disease detection, and labour management.

This project aims to solve real-world agricultural challenges by integrating Machine Learning with a user-friendly web interface.

---

## 🧠 Problem Statement

Farmers often rely on guesswork for:
- Crop selection
- Fertiliser usage
- Yield estimation
- Disease detection
- Labour management

This leads to unpredictable output and financial stress.

**Smart Agri-Tech** provides an intelligent and integrated solution to modernize farming decisions.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Samarth-J/Smart-Agri---Tech-Main-Project.git
cd Smart-Agri---Tech-Main-Project
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

Activate:

- Windows:
```bash
venv\Scripts\activate
```

- Mac/Linux:
```bash
source venv/bin/activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Setup Environment Variables

- Copy `.env.example`
- Create `.env`
- Add required keys

### 5️⃣ Run Application

```bash
python app.py
```

Open browser:

```
http://localhost:5000
```
